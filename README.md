Deskripsi Proyek

Proyek ini bertujuan untuk memprediksi kondisi medis dan lama rawat inap pasien dengan memanfaatkan Teknik machine learning. Analisis mendalam

dilakukan untuk mengidentifikasi faktor-faktor Utama yang mempengaruhi kondisi medis dan durasi rawat inap dan membangun model prediksi



Metodologi

EDA (Exploratory Data Analyst) :  distribusi LOS(Length Of Stay), disribusi kondisi medis

preprocessing : Encoding, Scaling, Feature Selection (SelectKBest)

Modeling : Model K-Nearest Neightbors(Klasifikasi) dan Random Forest (Regresi)



Rekomendasi Bisnis

* Memanajemen kapasitas ruangan untuk pasien dengan lama rawat inap Panjang (Kanker), sedang (Diebetes,Hypyetensi),Pendek("Healthy",Arthritis, dan Obesity).
* Menggunakan model machine learning regresi untuk memprediksi lama rawat inap pasien dengan memprediksi LOS(Length Of Stay) dapat memperkirakan beban ruang secara akurat
* Fokus pada indicator medis (Glucose,Oxygen Saturation, dll) untuk mengindentifikasi kecenderungan sebuah penyakit
* Menggunakan model machine learning klasifikasi untuk memprediksi secara akurat didukung dengan evaluasi lebih lanjut terhadap pola kondisi medis



Tools \& Libraries

Python,Jupyter Notebook,Scikit-learn,pandas,numpy,matplotlib,seaborn
